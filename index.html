Про програму

Програма реалізує утиліту ping мовою Python, яка імітує поведінку команди ping у середовищі Cisco Packet Tracer. Доступна у консольному (CLI) та графічному (GUI) варіантах. Працює на Windows, Linux та macOS.

Що таке ping?

Ping – мережева утиліта для перевірки доступності хоста. Надсилає пакети ICMP Echo Request та очікує ICMP Echo Reply. За часом відповіді визначається затримка (RTT) у мілісекундах.

Reply from 192.168.0.3: bytes=32 time<1ms TTL=128

Термінологічний словник

Основні поняття, що використовуються у програмі.

ТермінВизначення
IP-адреса Унікальна числова адреса пристрою в мережі. Наприклад, 192.168.1.1. Складається з 4 байтів (IPv4) від 0 до 255 через крапку. Використовується для ідентифікації відправника та отримувача пакетів.
Протокол Набір правил передачі даних у мережі. Визначає формат пакетів, порядок з'єднання та обробку помилок. Приклади: TCP, UDP, ICMP, HTTP, SSH.
Порт Числовий ідентифікатор (0–65535) для розрізнення служб на одному хості. Наприклад, :80 — HTTP, :443 — HTTPS, :22 — SSH. TCP-порт відкритий, якщо на ньому прослуховується служба.
ICMP Internet Control Message Protocol — службовий протокол для передачі повідомлень між мережевими пристроями. Саме його використовує команда ping: надсилається Echo Request, у відповідь приходить Echo Reply.
TCP Transmission Control Protocol — протокол з'єднання. Гарантує доставку пакетів у правильному порядку. Використовується HTTP, HTTPS, SSH, FTP. Перед передачею виконується «триетапне рукостискання».
RTT (Round-Trip Time) Час, за який пакет доходить від відправника до хоста і повертається назад. Вимірюється у мілісекундах (мс). Мале RTT — хороший зв'язок.
Підмережа / CIDR Спосіб розбиття IP-мереж на менші частини. Запис 192.168.1.0/24 означає, що перші 24 біти — адреса мережі (254 можливих хости).
Веб-сервіс / HTTP HTTP (HyperText Transfer Protocol) — протокол передачі вебсторінок. Порт 80 — незашифрований, порт 443 — зашифрований (HTTPS). Якщо порт відкритий — на хості працює вебсервер.
SSH Secure Shell — протокол безпечного віддаленого керування сервером через шифрований канал. Стандартний порт: 22. Якщо порт 22 відкритий — до хоста можна підключитись через SSH.
DNS Domain Name System — система, що перетворює доменні імена (наприклад google.com) в IP-адреси. Порт 53, протокол UDP/TCP.
Фільтрація трафіку Процес відбору мережевих пакетів за заданими критеріями: IP-адресою, протоколом або номером порту. Дозволяє обмежити або проаналізувати певний тип мережевої активності.
Брандмауер (Firewall) Програмний або апаратний засіб, що фільтрує мережевий трафік за правилами. Може блокувати окремі IP, порти або протоколи. Якщо порт закритий — можливо, його блокує брандмауер.
